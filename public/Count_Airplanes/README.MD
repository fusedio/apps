## Airplane Counter

This app counts airplanes in 5 airports, which come from predefined polygons.

The app runs the [UDF object detection across an AOI](https://www.fused.io/workbench/catalog/Airplane_Detection_AOI-71dc5cef-9fa7-42af-9e2b-d3c0f1320dbb) which in turn is calling the [UDF object detection inference (single tile)](https://www.fused.io/workbench/catalog/DL4EO_Airplane_Detection-d0d1cee6-f7fe-4f26-9a84-a7cb130d20cb).
